# CLI Beautification
## Instructions
- Use 'rich.table' for displaying task lists.
- Use 'rich.console' for color-coded status messages (Success=Green, Error=Red).
- Implement progress bars for long-running operations.
## Examples
User: "Show me my tasks."
Output: A formatted table with columns for ID, Priority (colored), and Status.
